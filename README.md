# Digital Solutions Agency 🚀

A high-performance digital agency landing page designed to help businesses establish a strong online presence and drive growth.

## 🔗 Live Demo
[https://digital-solutions-agency.vercel.app/](https://digital-solutions-agency.vercel.app/)

## 🛠 Tech Stack
- **Frontend:** React, Vite, Tailwind CSS
- **Deployment:** Vercel
- **Performance:** Optimized for speed and responsiveness

## 💡 Key Features
- **Modern UI/UX:** Clean and professional design tailored for business trust.
- **Fully Responsive:** Looks great on mobile, tablet, and desktop.
- **Fast Loading:** Built with Vite for instant page loads.
- **SEO Optimized:** Structured for better visibility.

## 🚀 Getting Started
1. Clone the repo: `git clone https://github.com/marzouki-ai/digital-solutions-agency.git`
2. Install dependencies: `npm install`
3. Start the project: `npm run dev`
